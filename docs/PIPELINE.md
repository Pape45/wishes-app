# Pipeline Documentation

## Overview
The CI/CD pipeline automates monthly objectives tracking and progress monitoring.

## Stages
1. **Display Objectives**
   - Shows monthly goals
   - Lists success criteria
   - Displays current status

2. **Evaluate Progress**
   - Checks completion status
   - Generates progress alerts
   - Provides timely reminders

3. **Generate Report**
   - Creates HTML report
   - Archives artifacts
   - Displays completion status

## Parameters
- `MONTH`: Month selection for wish tracking
- `DAY`: Current day (1-31)
- `ALL_OBJECTIVES_COMPLETED`: Mark if all objectives are completed

## Report Generation
- HTML report with status
- Progress visualization
- Objectives listing
